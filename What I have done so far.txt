=============================TECHNICAL SETUP=============================

Create a virtual environment with conda:
    - conda create --name togo python=3.6.3
Activate environment with conda:
    - conda activate togo
Install django inside the virtual environment:
    - pip install django
Setup the django project:
    - django-admin startproject togo

    - task name

    - username
